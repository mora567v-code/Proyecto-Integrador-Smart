# SmartCut Management System

## Integrantes
- Víctor Mora

## Objetivo del Sistema
Sistema de gestión para barberías desarrollado en Python. Permite registrar citas, controlar inventario de insumos, ver el reporte de caja diario y llevar el conteo de clientes atendidos.

## Descripción de Funcionalidades
- **Registrar Nueva Cita** – Ingresa el nombre del cliente y selecciona el servicio (Corte $10, Completo $18, Tinte $25, Manicura $12). Actualiza ganancias e inventario automáticamente.
- **Ver Inventario** – Muestra el stock actual de cera, shampoo, tintes y toallas con alertas de bajo nivel.
- **Reporte de Caja** – Muestra las ganancias totales acumuladas en la sesión.
- **Total de Clientes** – Contador de citas registradas en el día.
- **Salir** – Finaliza el sistema de forma segura.

## Tecnologías Utilizadas
- Lenguaje: Python 3
- Control de versiones: Git / GitHub
- Paradigma: Programación estructurada
